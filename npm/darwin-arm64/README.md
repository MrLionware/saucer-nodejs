# @aspect-build/saucer-nodejs-darwin-arm64

This is a prebuilt native binary for **saucer-nodejs** targeting **macOS ARM64 (Apple Silicon)**.

## Do not install directly

This package is automatically installed as an optional dependency of `saucer-nodejs` when you run:

```bash
npm install saucer-nodejs
```

## Supported Platforms

- macOS 11+ (Big Sur and later)
- Apple Silicon (M1, M2, M3, etc.)
- Node.js 20+

## Main Package

See [saucer-nodejs](https://www.npmjs.com/package/saucer-nodejs) for documentation and usage.
